String类型

2022/3/2

不可变性

字面量的赋值方式是在方法区划定内存区域，方法区内只会存一个值，因此，如果有2个连量指向同一个值，当其中一个变量被改变时，实际上是在内存区域重新划定一块区域给到这个变量；

通过new String("test")这种方式来造字符串，实际是造了2个对象，1个是存在heap区，另一个是在方法区划定内存存放“test“，将test的内存值赋值到heap区的对象上，heap对象的内存值给到stack的变量，

因此，两个变量都new String("test")，然后变量做 == 比较时，是false，原因就是两个变量的内存地址不一样。

字符串拼接

拼接时，若有变量参与，则等同于new 对象，则变量的地址值是来自于heap中；

若是纯字符串的拼接，则等同于字面量赋值，变量的地址值是来自于方法区；

String类方法

  • .length()，即char数组长度

  • .charAt()，获取字符串下标位的元素

  • .isEmpty()，长度是否为0

  • .toLowerCase()，.toUppderCase()，大小写转换

  • .trim()，去除字符串首尾的空格

  • .equalsIgnoreCase(String anotherString)

  • .concat(String str)，连接字符串

  • .compareTo(String anotherString)，比较两个字符串大小，实际则是用来按照字符a-z进行排序，若是负值，则前者小于后者，反之，前者大于后者

  • .substring(int beginIndex, int endIndex)
